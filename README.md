# Antiplag Print Report

Локальная копия страницы «Вывод отчёта в печать» сервиса [Antiplag](https://antiplag.uz/) — сохранена как отдельный статический проект со всеми ресурсами (стили, скрипты, изображения).

## Структура проекта

```
antiplag-print-report/
├── index.html              # страница отчёта
├── assets/
│   ├── css/
│   │   ├── print.css           # основные стили печатной формы отчёта
│   │   ├── jquery-ui.min.css   # стили jQuery UI
│   │   └── google-fonts.css    # шрифт Open Sans (Google Fonts)
│   ├── js/
│   │   ├── print.js            # логика круговой диаграммы и подсветки цитат
│   │   └── print-jquery.js     # библиотека jQuery
│   └── images/
│       ├── antiplag-logo.png   # логотип Antiplag
│       └── report-preview.png  # изображение из отчёта
└── README.md
```

## Запуск

Просто откройте `index.html` в браузере — все ресурсы подключены относительными путями, интернет не требуется (кроме шрифтов Google Fonts, если хотите, чтобы они подгружались онлайн).

## Примечание

Это статический снимок страницы (сохранённая веб-страница), а не исходный код сервиса Antiplag. Оригинальные пути к файлам (абсолютные, вида `C:\Users\...`) были заменены на относительные, чтобы проект корректно открывался на любой машине.
